Routing Technologies Tested in CCIE ENCOR Exam

1. Routing Protocols (30%)

This domain covers the core routing protocols and their configurations in enterprise environments:

  • Open Shortest Path First (OSPF):
    • OSPFv2 and OSPFv3: Configuration, operation, and troubleshooting of OSPF in IPv4 and IPv6 networks.
    • OSPF Areas: Implementing single-area and multi-area OSPF designs, including backbone area (Area 0) and non-backbone areas.
    • OSPF Authentication: Configuring OSPF authentication to secure routing updates.
    • OSPF Route Summarization: Implementing and troubleshooting route summarization to reduce the size of routing tables.
    • OSPF Virtual Links: Configuring virtual links for OSPF adjacency across non-contiguous areas.
    • OSPF LSA Types: Understanding and managing OSPF Link State Advertisements (LSAs), including Type 1 (Router LSA), Type 3 (Network LSA), and Type 5 (External LSA).
  • Enhanced Interior Gateway Routing Protocol (EIGRP):
    • EIGRP Configuration and Operation: Configuring EIGRP for both IPv4 and IPv6 networks.
    • EIGRP Metrics: Understanding and adjusting EIGRP metrics (bandwidth, delay, load, reliability).
    • EIGRP Stub Routing: Configuring and using EIGRP stub routing to reduce network traffic in certain topologies.
    • EIGRP Named Mode: Implementing EIGRP in named mode to simplify configuration and improve manageability.
  • Border Gateway Protocol (BGP):
    • EBGP and IBGP Configuration: Configuring and troubleshooting EBGP (external BGP) and IBGP (internal BGP) for both IPv4 and IPv6.
    • BGP Attributes: Understanding and manipulating BGP path selection using attributes like AS path, local preference, MED (multi-exit discriminator), and community.
    • BGP Route Filtering: Implementing route filtering techniques such as prefix lists, route maps, and AS path access lists.
    • BGP Path Selection: Adjusting BGP path selection rules using various methods such as local preference, weight, and MED.
    • BGP Multihoming: Configuring BGP for multihoming to provide redundancy and load balancing between different service providers.
  • Static Routing:
    • Basic Static Routing: Configuring static routes for IPv4 and IPv6.
    • Floating Static Routes: Implementing floating static routes for backup routes when dynamic routing protocols fail.
    • Default Routing: Configuring default routes to handle traffic destined for unknown networks.
  • Routing between VLANs (Inter-VLAN Routing):
    • Router-on-a-Stick: Configuring subinterfaces on a router to enable routing between multiple VLANs.
    • SVI (Switched Virtual Interface): Configuring SVIs on Layer 3 switches to provide inter-VLAN routing.

2. Routing Redistribution (15%)

Redistribution involves sharing routing information between different routing protocols. The exam covers:

  • Redistributing Routes Between Protocols: Configuring and troubleshooting redistribution between different routing protocols such as OSPF, EIGRP, and BGP.
  • Route Maps and Filters: Using route maps to control redistribution and prevent undesirable routing loops or suboptimal paths.
  • Metric Adjustment: Adjusting metrics when redistributing routes to ensure correct routing decisions are made.
  • Redistribution Loops: Preventing routing loops during redistribution using techniques like route filtering, tag-based redistribution, and route maps.

3. Routing Policy and Path Control (15%)

Path control techniques allow for fine-grained control over routing decisions. Key topics include:

  • Policy-Based Routing (PBR): Configuring and troubleshooting policy-based routing to influence traffic flow based on criteria other than destination IP.
  • BGP Path Manipulation: Implementing BGP path manipulation techniques, including adjusting AS path, weight, local preference, and MED.
  • Prefix Lists: Creating and using prefix lists for route filtering in various routing protocols.
  • Route Maps: Using route maps for advanced routing policies such as route filtering, redistribution, and path control.

4. Multicast Routing (10%)

Multicast routing protocols and their configuration are essential for efficient data distribution in enterprise networks:

  • Protocol Independent Multicast (PIM): Configuring PIM (Sparse Mode, Dense Mode, and Source-Specific Multicast) to enable multicast routing.
  • PIM Sparse Mode (PIM-SM): Configuring and troubleshooting PIM-SM for efficient, scalable multicast routing.
  • IGMP (Internet Group Management Protocol): Understanding and configuring IGMP for managing multicast group memberships.
  • Multicast Routing with BGP: Configuring BGP for multicast routing, particularly for large-scale implementations.

5. High Availability and Redundancy (10%)

The CCIE ENCOR exam tests knowledge of ensuring high availability in routing:

  • First Hop Redundancy Protocols (FHRP): Configuring and troubleshooting HSRP (Hot Standby Router Protocol), VRRP (Virtual Router Redundancy Protocol), and GLBP (Gateway Load Balancing Protocol).
  • Routing Protocol Failover: Understanding and configuring routing protocol failover mechanisms, such as EIGRP or OSPF route redistribution in case of link failures.
  • Dual-Homed Connectivity: Implementing redundancy between routers or networks for fault tolerance and high availability.
  • Equal-Cost Multi-Path (ECMP): Configuring and optimizing ECMP for load balancing between multiple routes with equal cost.

6. IPv6 Routing (10%)

IPv6 is essential for modern routing infrastructure. The exam tests the following:

  • IPv6 Routing Protocols: Configuring and troubleshooting OSPFv3, EIGRP for IPv6, and BGP for IPv6.
  • Static Routing with IPv6: Configuring IPv6 static routes and floating static routes.
  • IPv6 Addressing: Understanding IPv6 addressing, including global unicast, link-local, and multicast addresses.
  • IPv6 Tunneling: Configuring IPv6 tunneling protocols like GRE and 6to4 to facilitate IPv6 communication across IPv4 networks.

7. Network Optimization and Troubleshooting (5%)

Network optimization and troubleshooting are key components of the exam:

  • Traffic Engineering: Configuring and troubleshooting MPLS Traffic Engineering (MPLS-TE) for optimal bandwidth utilization.
  • Troubleshooting Routing Protocols: Identifying and resolving issues in routing protocols, including OSPF, EIGRP, BGP, and IPv6 routing.
  • Path MTU Discovery: Configuring and troubleshooting Path MTU Discovery (PMTUD) to avoid fragmentation in routing.
